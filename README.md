## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Merssith/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Generate the .env file, using as a base the example found in this repository
4. Run the app (for testing)
   ```sh
   npm run dev
   ```
